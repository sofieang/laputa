Simulations

A simulation is an evolution of a society. These are controlled through the floating window "Simulation". This window contains buttons to step the simulation forward one step at a time as well as to start and stop continuous simulations. The pop-up menu "Log" is used to control what amount of information to display when running a simulation.

When running or stepping simulations, statistics is collected:

Time: The number of time steps elapsed since the "Reset" button was last pressed.

Error avg.: The current average degree of error in the society.

Error dev.: The current standard deviation of the degree of error in the society.

Error Δ: How much the average degree of error has changed since the "Reset" button was last pressed.

Trust avg.: The average expected trust for all links in the society (note: trust in inquiry is not included here).

Trust Δ: How much the average expected trust of all links has changed since the "Reset" button was last pressed.
