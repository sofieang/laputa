Probability Distributions

Probability distributions are used more or less everywhere in Laputa, and come in two kinds: distributions, which determine single values, and metadistributions, which determine entire probability distributions (most importantly, trust functions).

Distributions

Distributions are referenced by name to facilitate reuse. Clicking on any distribution control allows the selection of a distribution to govern the values of the property in question. From this menu, it is also possible to create new distributions, and make copies of existing ones. Distributions that have been created (i.e. not the default distributions) can also be edited.

For editing and creating new distributions, the distribution dialog is used. This dialog contains an area where the distribution is displayed and can be changed, as well as sliders for parameters, and a pop-up menu to set the distribution's type:

Point: Always gives the same value, set by the slider or by clicking in the display.

Interval: Gives a value in the interval indicated, with any value in that interval being equally likely.

Normal: Gives a value randomized from a normal distribution. The mean and standard deviation are adjustable.

Beta: Gives a value randomized from a beta distribution. The parameters alpha and beta are adjustable.

Freeform: Allows the user to draw a distribution of her own.

There are also buttons for editing distributions numerically, as well as for saving changes. Since distributions are name referenced, no two distributions can be saved with the same name.

Metadistributions

Metadistributions are distributions over distributions. Since a distribution, in itself, is such a high-dimensional object, Laputa uses a highly simplified approach to metadistributions to make them manageable. Essentially, all metadistributions are interval distributions. A distribution is generated from a metadistribution by rolling a random number in [0, 1]. The Amount Distribution determines how this value is rolled: setting it to an interval distribution [x, y] makes the random number rolled be picked uniformly from the interval [x, y], for instance.

The random number rolled is then used to create a mixture of the distributions specified as 'One' and 'Zero'. Both of these must be of the same type, and the blending proceeds by creating a new distribution whose parameters have values that are weighted averages of the parameters of 'One' and 'Zero'. The weight of 'One', in this case, is the random number obtained from the Amount distribution, and the weight of 'Zero' is one minus this value.

On the left-hand side, a selection of mixtures is visible, given the current settings. This is intended to give a feel for the kind of distributions producible by the metadistribution in question.
