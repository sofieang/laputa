Inquirers

The properties of an inquirer are accessed through the Inquirer Inspector, which is a floating window.

Name: An optional name for the inquirer. Does not affect anything except aesthetics.

Belief: The current degree of belief the inquirer has in the sentence whose truth-value is investigated.

Inquiry Chance: The chance to engage in inquiry each time step.

Inquiry Accuracy: The chance to that inquiry gives the correct result (i.e. p) rather than its opposite (i.e. not-p).

Inquiry Trust: The trust function that describes how much the inquirer trusts the results of her own inquiry. The accompanying popup menu can be used to set this function to a number of presets, as well as to an arbitrarily specified probability distribution. The trust function can also be edited manually by drawing or clicking in the display. A check box allows switching the inquirer's update of trust in view of the results of inquiry on or off.

Simulation Parameters: Accesses specific parameters that determine how the inquirer's properties should vary in a batch simulation. These are described in the text simulation parameters.
