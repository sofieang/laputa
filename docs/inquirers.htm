Inquirers

An inquirer is any agent in Laputa, except Nature. Thus an inquirer does not need to engage in inquiry, or for that matter communication, although it can do either or both of these. Inquirers, as created in a society, come in two variants: concrete inquirers, and template inquirers.

A concrete inquirer has the properties she has, which are given in the Inquirer Inspector, and described below. A template inquirer also has an associated set of simulation parameters, which determine the inquirer's properties in batch simulations. The template inquirers can thus be interpreted as standing for a certain role in a society, rather than a specific person.

The concrete properties of an inquirer are accessed through the Inquirer Inspector, which is a floating window. When more than one inquirer is selected, the values shown are the averages (or in the case of Sources and Listeners, the totals) of the individual inquirers in the selection.

Name: An optional name for the inquirer. Does not affect anything except aesthetics.

Sources: The number of links going to this inquirer.

Listeners: The number of links going from this inquirer.

Belief: The current degree of belief the inquirer has in the sentence whose truth-value is investigated.

Inquiry Chance: The chance to engage in inquiry each time step.

Inquiry Accuracy: The chance to that inquiry gives the correct result (i.e. p) rather than its opposite (i.e. not-p).

Inquiry Trust: The trust function that describes how much the inquirer trusts the results of her own inquiry. Clicking on it allows the user to set it to any probability distribution, although, internally, a trust function is always stored as a freeform distribution with a finite resolution. A check box allows switching the inquirer's update of trust in view of the results of inquiry on or off.

In Statistics: If off, the inquirer is not included when calculating a society's E-value

Is Template: If on, this is a template inquirer rather than a concrete one. The associated Parameters button allows the user to edit the simulation parameters of the inquirer.
