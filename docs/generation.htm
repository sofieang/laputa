Generating Random Societies

The interface for this is much the same as that for batch simulations, since both require setting a society's properties randomly. Just generating societies does not require stages, however, and the dialog contains only the following tabs:

  • Network
  • Inquriers
  • Links

As in the batch simulation window, the generate society dialog can be used either to generate new societies, or to change the values of the current society randomly, depending on whether the Create check boxes in the network tab are ticked or not.

The inquirers of the generated society will be placed according to their degrees, so that the ones with the highest degree (i.e. incoming and outgoing links) are in the center, and such that inquirers with lower degrees are farther out.
