The Expressions dialog

This dialog lets one set a freeform distribution to the values of a mathematical expression. Laputa uses the muParser library to parse the expression, which must be given as a function of t. The function does not have to be normalised, and any negative values it takes will be interpreted as zero.

Built-in operators and functions that are supported include the following, as taken from the parser's webpage.
