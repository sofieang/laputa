The Record tab

In this tab, what data to record during the batch simulation can be adjusted. The reason for doing this is mostly to avoid problems with memory when doing very large simulations. If E-Value data is not recorded, only rudimentary statistics will be available in the statistics dialog. If topology is not recorded, it will not be possible to export network topologies from the export topologies dialog after the simulation has run.

In the accompanying fields, it is possible to change how often data about E-Values and topologies is to be recorded. Raising these values will also lessen the amount of memory Laputa will need to run the simulation.
