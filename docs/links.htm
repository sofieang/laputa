Links

A link describes how one inquirer may recaive information from another, and how she treats this information. Like inquirers, links come in concrete and template> versions. Also, like the inquirer, a template link is interpretable as a type of link, raher than a specifically determined one. The properties of a link are accessed through the Inquirer Inspector, which is a floating window.

Listen Chance: The chance that the link is used to ask whether or not p each time step.

Threshold: When above 0.5, the least degree of belief in p the source must have, in order to claim that p. When below 0.5, the highest degree of belief in p the source can have, in order to falsely claim that p. If set to exactly 0.5, the inquirer's decision of whether to say p or not-pwill be purely random.

New Evidence Requirement: What kind of new evidence the source of this link must have received since the link was last used in order to use it again:

None: No new evidence is required; source can always say what she wishes, even if she has received no new evidence.
Inquiry: Source must have received some result from her own inquiry since she communicated over this link last in order to use it.
Any: Source must have received either some result from her own inquiry, or information from one of her other sources (excluding the destination inquirer of the link) since she communicated over this link last in order to use it.
There is also a check-box for controlling how to handle the prior beliefs (i.e. those the inquirer starts with):
Treat Prior as Fulfilling Requirement: If ticked, the first instance of communication over this link is exempted from the "new evidence" requirement. This in effect allows the source to communicate just based on her prior.

Trust: The trust function that describes how much the inquirer trusts the source. Clicking it allows the user to set it to an arbitrarily specified probability distribution. A check box allows switching the inquirer's update of trust in view of the information received on or off.

Is Template: If on, this link is a template rather than a concrete link. The template's properties are set through the simulation parameters dialog accessible through the Parameters button
