Simulation Parameters

These parameters, for links and inquirers, determine how their proprties should be set when the batch simulator creates a new society. For each property, there are three possible settings:

Default: the property is set entirely depending on the settings in the Batch Simulation window.

Constant: the property does not change randomly at all, i.e. the inquirer/link has the same value in every random society, and does not depend on the settings in the Batch simulation window.

Variable: the property varies individually, rather than based on the settings in the Batch simulator. How it is varied is determined by the probability distribution next to the setting knob.

For inquirers, it is also possible to switch on or off whether data from this inquirer should be included in the statistics collected in the simulation window, and in the batch simulator.
