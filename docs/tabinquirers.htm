The Inquirers tab

This tab determines the individual properties of each inquirer at the start of the stage. Each property has a probability distribution (or in the case of trust, a metadistribution), and a check box that determines whether to change this property at the start of the stage, or let it stay as it was at the end of the previous stage.

The properties that can be modified are those of concrete inquirers, as described in the corresponding page.
